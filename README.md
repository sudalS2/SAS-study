* 변수명 규칙 
1. 첫 문자로는 반드시 영문자나 밑줄(_) 이용 
2. 두 번째 문자부터는 영문자, 밑줄, 숫자 사용 가능
3. 이름 중간에 빈칸이 있을 수 없음
